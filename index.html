Data Scraping SNPMB

Tugas Besar Proyek 1 Kelompok B1

Deskripsi

touch_app

Tujuan

Website yang kami rancang dari hasil web scraping website resmi SNPMB ini menyajikan data komprehensif dari setiap program studi di perguruan tinggi negeri di Indonesia berdasarkan data SNPMB.

Manfaat

Dengan tujuan memberikan informasi yang akurat dan terkini, website ini dirancang untuk membantu calon mahasiswa, orang tua, dan konselor pendidikan dalam mengambil keputusan yang lebih baik mengenai pilihan program studi dan universitas.

Harapan

Proses seleksi dan penerimaan mahasiswa baru menjadi lebih transparan dan terinformasi, sehingga meningkatkan kesesuaian antara minat, bakat, dan kapasitas di perguruan tinggi negeri Indonesia.

table
Tabel

Website ini dilengkapi dengan tabel yang merinci jumlah peminat dan daya tampung setiap program studi di perguruan tinggi negeri selama 5 tahun terakhir, memberikan gambaran lengkap tentang tren penerimaan.

bar_chart
Visualisasi

Website ini menyediakan visualisasi data dalam bentuk chart dan grafik interaktif, memudahkan pengguna untuk memahami dan menganalisis data secara lebih intuitif dan efisien.

Pencarian Informasi

Fitur pencarian interaktif pada website ini memungkinkan pengguna untuk menemukan informasi spesifik mengenai program studi atau universitas dengan cepat dan mudah
